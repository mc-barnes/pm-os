## Design Controls Readiness
- [ ] DHF index exists and is current (regulatory/design-history/)
- [ ] User needs documented with acceptance criteria
- [ ] Design inputs traceable to user needs (UN → DI)
- [ ] Design outputs traceable to inputs (DI → DO)
- [ ] Verification records with quantitative pass criteria (DO → VER)
- [ ] Validation records under simulated/actual use (UN → VAL)
- [ ] Traceability matrix complete — no orphan rows
- [ ] Design changes have change control records

## Risk Management Readiness
- [ ] Risk management plan exists and is current
- [ ] Risk acceptability matrix defined and version-controlled
- [ ] Hazard identification complete (all foreseeable hazards)
- [ ] Risk estimation uses defined scales (not ad hoc)
- [ ] Risk controls implemented and verified
- [ ] Residual risk evaluated — none remain Unacceptable
- [ ] Overall residual risk benefit-risk conclusion documented
- [ ] Risk management report complete
